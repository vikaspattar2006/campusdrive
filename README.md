# CampusDrive

A project designed to manage and track campus placement activities efficiently.

## 🚀 Features
- Store student details
- Track placement status
- Manage company drives
- Organize campus recruitment data

## 🛠️ Tech Used
- C++ (or update if different)

## ▶️ How to Run
1. Compile:
   g++ main.cpp -o campusdrive
2. Run:
   ./campusdrive
